Local GitHub Workflow With Command-Line Git & Docker

Tutorial Overview


The purpose of this tutorial will be to explore how to incorporate command-line Git into your local workflow. I will demonstrate my workflow by building an html feature that will be pushed to a production repository. I will also test my work locally with super-linter from the the command-line using docker.

  • I'll share the software development tools I use in conjunction with GitHub that make my workflow possible.

  • I'll explore forking a repo that you want to contribute to.

  • I'll explore how to set up a local environment to work with a GitHub repo.

  • I'll explore the workflow for managing code edits and keeping your work in sync with GitHub.

  • I'll explore how to test your code locally both in a browser and using docker to run super-linter

  • I'll explore how to merge my code edits back to the production code on GitHub
