How did we change 'websitedemo'?

We started with a different website and thought about what would be the most helpful to us and as well as other students. We decided that a site that could be a resource for students learning about Git and GitHub would be the best. It would show that we know the material and that we can help answer questions. It also made us aware as we were approaching each step in the process, that we would need to be able to relate that to others, some with Information Technology experience and some who have made simple web pages in the past.
